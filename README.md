## Ввод данных с клавиатуры
В Python, как и в любом другом языке программирования, значения переменным можно присваивать напрямую в коде, например:
```Python
his_name = 'Oliver'
her_name = 'Alisa'

his_age = 19
her_age = 18
```

Однако, мы также можем предоставить пользователям возможность вводить значения самостоятельно, используя функцию `input`. Например:
```Python
his_name = input("Введите имя для него: ")
her_name = input("Введите имя для неё: ")

his_age = int(input("Введите его возраст: "))
her_age = int(input("Введите её возраст: "))
```
В данной конструкции, мы/пользователь сам сможет ввести данные с клавиатуры. Заметьте, что при этом указывается какой тип дынных мы вводим.

**Таблица типов данных с `input`**
| Функция Input       | Описание                                          | Тип данных       | Пример ввода                          |
|---------------------|---------------------------------------------------|------------------|---------------------------------------|
| `input()` или `str(input())`          | Стандартный ввод, возвращает строку               | `str`            | `name = input("Введите имя: ")`       |
| `int(input())`      | Преобразует ввод в целое число                    | `int`            | `age = int(input("Введите возраст: "))` |
| `float(input())`    | Преобразует ввод в число с плавающей точкой       | `float`          | `height = float(input("Введите рост: "))` |
| `bool(input())`     | Преобразует ввод в булевое значение (`True/False`)| `bool`           | `is_student = bool(input("Студент? (True/False): "))` |
| `list(input())`     | Преобразует строку в список символов              | `list`           | `chars = list(input("Введите символы: "))` |
| `tuple(input())`    | Преобразует строку в кортеж символов              | `tuple`          | `chars = tuple(input("Введите символы: "))` |
| `eval(input())`     | Выполняет строку как код Python                   | Разные типы      | `value = eval(input("Введите выражение: "))` |

### Примечания:
- `input()` всегда возвращает строку, поэтому для получения других типов данных необходимо применять преобразование.
- `eval(input())` следует использовать с осторожностью, так как это может быть небезопасно и потенциально выполнить нежелательный код.
- `bool(input())` возвращает `True` для любых непустых строк, а `False` только для пустых строк.
- `input()` и `str(input())` - одинаковые записи, возвращается строка.

## Способы вывода данных с помощью `print()`

[![Typing SVG](https://readme-typing-svg.herokuapp.com?font=Fira+Code&pause=1000&color=488A4B&width=435&lines=print(f'Hello+%7Bname%7D!');print('Hello'%2C+name%2C+'!');print('Hello+%7B0%7D!'.format(name)))](https://git.io/typing-svg)

**Примеры различных способов использования функции print() для вывода данных в Python**

В Python можно выводить данные различными способами. Рассмотрим три основных подхода: стандартный вывод через запятую, с использованием `format()` и через f-строки.

### 1. Стандартный вывод через запятую

При использовании запятой в `print()` значения выводятся через пробел:

```python
name = "Игорь"
age = 19
city = "Москва"

# Пример вывода:
print("Имя:", name, "Возраст:", age, "Город:", city)
```
Выходные данные:
```
Имя: Игорь Возраст: 19 Город: Москва
```

### 2. Вывод с использованием метода `format()`

Метод `format()` позволяет вставлять значения переменных в строку с помощью `{}`. Он поддерживает нумерацию и именованные аргументы.

```python
name = "Игорь"
age = 19
city = "Москва"

# Пример вывода:
print("Имя: {}, Возраст: {}, Город: {}".format(name, age, city))
# Также можно использовать нумерацию или именованные аргументы:
print("Имя: {0}, Возраст: {1}, Город: {2}".format(name, age, city))
print("Имя: {n}, Возраст: {a}, Город: {c}".format(n=name, a=age, c=city))
```
Выходные данные:
```
Имя: Игорь, Возраст: 19, Город: Москва
```

### 3. Вывод с использованием f-строк

f-строки (или интерполяция строк) появились в Python 3.6 и позволяют вставлять значения переменных прямо в строку, используя `{}` и префикс `f` перед строкой.

```python
name = "Игорь"
age = 19
city = "Москва"

# Пример вывода:
print(f"Имя: {name}, Возраст: {age}, Город: {city}")
```
Выходные данные:
```
Имя: Игорь, Возраст: 19, Город: Москва
```
Каждый из способов имеет свои преимущества: 

* Стандартный вывод через запятую прост и удобен для базовых задач. 
* format() позволяет более гибко работать с форматированием строки.
* f-строки делают код более читаемым и понятным, особенно при использовании переменных.

## Ещё элемент форматирования 

[![Typing SVG](https://readme-typing-svg.herokuapp.com?font=Fira+Code&pause=1000&color=CF3E6B&width=435&lines=print(%22First+line%5CnSecond+line%22);print(f%22%7Bname%7D%5Cn%7Bage%7D%22))](https://git.io/typing-svg)

Разберём один из элементов форматирования в Python - `\n` <br>
`\n` в Python — это управляющий символ, который используется для перехода на новую строку при выводе текста. Он часто применяется в строках и выводе данных, чтобы структурировать текст и сделать его более читаемым.

**Основные моменты про `\n`**

* `\n` называется переносом строки или новой строкой.
* Работает во всех строках, где бы он ни использовался: в `print()`, при работе со строками, в файлах и т.д.
* Он вставляет в текст новый строковый разделитель, который заставляет последующий текст начинаться с новой строки.

## Примеры использования

### 1. Использование в `print()`

```python
print("Первая строка\nВторая строка\nТретья строка")
```
Выходные данные: 
```
Первая строка
Вторая строка
Третья строка
```

### 2. Использование в строках

```python
text = "Hello\nWorld\nPython"
print(text)
```
Выходные данные: 
```
Hello
World
Python
```

### 3. Использование с f-строками и format()

```python
name = "Игорь"
age = 19
city = "Москва"

# Использование \n с f-строкой
print(f"Имя: {name}\nВозраст: {age}\nГород: {city}")

# Использование \n с format()
print("Имя: {}\nВозраст: {}\nГород: {}".format(name, age, city))
```
Выходные данные: 
```
Имя: Игорь
Возраст: 19
Город: Москва
```
